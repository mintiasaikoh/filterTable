# フィルターテーブル — Power BI カスタムビジュアル

テキスト検索（含む／含まない）で行を絞り込み、チェックボックスで他のビジュアルをクロスフィルタリングできるテーブルビジュアル。

## 機能

| 機能 | 詳細 |
|---|---|
| テキスト検索 | 「を含む」「を含まない」の条件を複数組み合わせて AND / OR で絞り込み |
| 全件検索 | `AdvancedFilter` で PBI クエリエンジンに条件を渡すため、100k 件超のデータでも全件対象 |
| 行選択・クロスフィルター | チェックボックスで行を選択すると `BasicFilter` を発行し、同ページの他ビジュアルをフィルタリング |
| スライサーの同期 | Power BI の「スライサーの同期」パネルに対応。複数ページ間での連携が可能 |
| 列タブ | 複数列がある場合は列名チップで表示列を切り替え（「全列」で全表示） |
| 仮想スクロール | DOM に描画するのは可視行のみ。大量データでもスムーズにスクロール |
| 状態の永続化 | フィルター条件・論理演算子・適用済み条件をレポートファイルに保存 |

## 使い方

### フィールドの設定

「列」フィールドウェルに任意の列を追加します。複数列追加可能です。

### フィルター操作

1. **条件を追加** — `+ 条件を追加` ボタンで条件行を追加
2. **列・演算子・キーワードを入力** — 列を選択し、「を含む」または「を含まない」を選んでキーワードを入力
3. **AND / OR を選択** — 複数条件の結合論理をヘッダーのトグルで切り替え
4. **実行** — `実行` ボタンまたは Enter キーで検索を適用
5. **解除** — `解除` ボタンで全件表示に戻す

### 行選択

- 行左端のチェックボックスをオンにすると、その行の値で他ビジュアルをフィルタリング
- 複数行選択可能（ヘッダーのチェックボックスで表示中の全行を一括選択）
- ステータスバーの `選択解除` でクリア

### 列タブ

複数列追加時にビジュアル上部にチップが表示されます。チップをクリックするとその列だけを表示します。「全列」で全列表示に戻ります。

## 検索と行選択の排他動作

**テキスト検索**と**行選択によるフィルター**は同じフィルタースロットを使用するため排他です。

- `実行` を押すと行選択はリセットされ、検索条件が PBI エンジンに送られます
- チェックボックスを操作すると検索フィルターが上書きされ、選択行の値で BasicFilter が発行されます

## 100k 件超のデータについて

`dataReductionAlgorithm` の上限（100,000 行）を超えるデータに対しては、`AdvancedFilter` を利用して PBI のクエリエンジン側で検索を実行します。ビジュアルには検索結果の行だけが送られてきます。

ただし以下のケースは in-memory 検索にフォールバックします（ビジュアルが受け取った行の中だけで絞り込まれます）。

- 複数列にまたがる **OR** 条件（PBI の `AdvancedFilter` では表現不可）
- 同一列に **3 条件以上**ある場合の 3 件目以降（PBI 側で先頭 2 件を絞り、残りを in-memory で絞る）

## 開発

### 必要環境

- Node.js 18+
- `powerbi-visuals-tools` (`pbiviz`)

```bash
npm install
```

### 開発サーバー起動

```bash
npm start
```

Power BI のレポートで開発者ビジュアルを有効にして動作確認します（テナント設定 → 開発者向け設定）。

### パッケージ生成

```bash
npm run package
```

`dist/filterTableVisual.1.0.0.0.pbiviz` が生成されます。Power BI の「カスタムビジュアルのインポート」からファイルを読み込んでください。

## ファイル構成

```
src/
  visual.ts        メインロジック
  settings.ts      フォーマットモデル設定
style/
  visual.less      スタイル
capabilities.json  データロール・フィルタースロット定義
pbiviz.json        ビジュアルメタデータ
```

## 技術仕様

| 項目 | 値 |
|---|---|
| API バージョン | 5.3.0 |
| データ取得上限 | 100,000 行（`dataReductionAlgorithm: top`） |
| 行の高さ | 24px（仮想スクロール計算用定数） |
| 仮想スクロールバッファ | 上下 8 行 |
| デバウンス（入力保存） | 800ms |
